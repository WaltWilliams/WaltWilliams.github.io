Class Final Projects

Below are my class final projects in chronical order. The further I got in my classes the more I found I enjoyed the back end mechanics more that creating GUI user interfaces.

  • CS-1400 - Fundamentals of Programming - Southern Utah University.
    Cryptic is a One Time Pad implementation with a Swing user interface, and uses a singly circular linked list for the encryption wheel - See repo for detail on how this works.
    View the code and details at the One Time Pad repo.

  • CS-1410 - Object Oriented Programming - Southern Utah University.
    Rhesymeg is a Truth Table generator. Enter a logical proposition like [(A∨B)∙(B⊃A)] and this program will produce a truth table for it.
    View the code at Rhesymeg repo.

  • CS-2420 - Introduction to Algorithms and Data Structures - Southern Utah University.
    Slide Show is a slide show program. It was a joint effort with two other class mates which I collaborated with on work in several of my classes.
    View the code at the Slide Show repo.

  • CS-3000 - C and C++ Programming - Southern Utah University.
    3 sorting algorithms. Counting Sort, Radix Sort, Bucket Sort. This time done in PHP
    View the code at the Sorting repo.

  • CS-3150 - C and C++ Programming - Southern Utah University.
    Mystery Market is a C++ program that makes use the queue class from the STL. It is setup to simulate a supermarket check stand. It will run for 10 shoppers and then terminate.
    View the code at the Mystery Market repo.
