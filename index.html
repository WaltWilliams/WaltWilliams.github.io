Post College Projects

I have not ceased to study and work on projects.

  • MyClassicCar
    An Android app for classic car owner. It primary feature is storing part numbers for difficult to locate parts. It also tracks oil changes, maintenance and repairs. It makes use of the RecyclerView, ListView, and several other Android elements. All persisted with the SQLite DBMS.
    Available now on Google Play.

  • DigiCalc
    An Android app that calculates 64 bit binary, octal and hexadecimal values. It makes use of Tabs, Fragments and alternate layouts for when the device is rotates and layouts for tablets.
    Available now on Google Play.
    View the code at the DigiCalc GitHub repo.

  • Kryptic
    A Python 3 and TkInter program that performs a One-Time-Pad encryption. The text to encrypted and the "Pad" file are read in from a files. The main part of the process is emulated with a singly circular linked list. Go to the GitHub repository (link below) for details on the One-Time-Pad encryption process.
    View the code at the Kryptic Github repo.

  • ShowTime
    A Slideshow program program writte in C++ featuring a Qt user interface.

Working With Microcoltrollers

Perhaps may not be brilliantly spectacular but it sure is a lot of fun. One of the Microchip microcoltrollers and one for the Arduino. My interest for this stems from a College class I took called Applications of Microprocessors. The class work for this class was done in MPASM Assembly. Microchip has since deprecated MPASM and gone exclusively with XC8, which means I'm starting back square-one. I decided to learn the C programming first and then begin work with the XC8 Assembly.
