My Projects

Continuing to study and work on projects.

  • MyClassicCar
    An Android app for classic car owner. Its primary feature is storing part numbers for difficult to locate parts. It also tracks oil changes, maintenance and repairs. It makes use of the RecyclerView, ListView, and several other Android elements. All persisted with the SQLite DBMS.
    Available now on Google Play.

  • DigiCalc
    An Android app that calculates 64 bit binary, octal and hexadecimal values. The app makes use of Tabs, Fragments and alternate layouts for when the device is rotated as well as layouts for tablets. The idea for this app came while furthering my studies of computer architecture.
    Available now on Google Play.
    View the code at the DigiCalc GitHub repo.

  • Kryptic
    A Python 3 and TkInter program that performs a One-Time-Pad encryption. The text to be encrypted and the "Pad" are both read in from files. The main part of the process is emulated with a singly circular linked list. Go to the GitHub repository (link below) for details on the One-Time-Pad encryption process.
    View the code at the Kryptic Github repo.

  • ShowTime
    A Slideshow program written in C++ featuring a Qt user interface. This program makes use of threads. Most notably Qt's version of them.
    View the code at the Kryptic GitHub repo.
