Project Description

Online employee scheduling system

This web application has the following features:
  • Sharing a single employee with up to three store departments.
  • Start schedules on any day of the week.
  • Automated time off request form.
  • Prevent scheduling errors like double scheduling of shared employees.
  • Overtime warning.
  • Adjustable department specific settings.
  • Several adjustable system wide settings.
  • Schedules will be viewable on the Internet from anywhere.


I reserved the domain name of WoodlandCoders.com. I am not able to have the application available here for a try-out because Google-Drive is not capable of hosting programs which require PHP and MariaDB web server modules, and I currently cannot afford a hosting service.
