Class Final Projects

Each of the coding classes at Southers Utah University had a "Final Project". Below are my class final projects in chronical order. The further I got in my classes the more I found I enjoyed the back end mechanics more that creating GUI user interfaces.

  • Cryptic
    A One Time Pad implementation with a Swing user interface, and uses a singly circular linked list for the encryption wheel - See repo for detail on how this works. These files are setup in an IntelliJ IDEA (Community Edition) project for easy viewing.
    View the code and details at the One Time Pad repo.

  • Rhesymeg
    A Truth Table generator. Enter a logical proposition like [(A∨B)∙(B⊃A)] and this program will produce a truth table for it. These files are setup in an IntelliJ IDEA (Community Edition) project for easy viewing.
    View the code at the Rhesymeg repo.

  • Slide Show
    A slide show program. It was a joint effort with two other class mates which I collaborated with on work in several of my classes. These files are setup in an IntelliJ IDEA (Community Edition) project for easy viewing.
    View the code at the Slide Show repo.

  • 3 Sorting Algorithms
    Counting Sort, Radix Sort, Bucket Sort. This time done in PHP
    View the code at the Sorting repo.

  • Mystery Market
    A C++ program that makes use the queue class from the STL. It is setup to simulate a supermarket checkstand. The checkstand being the queue. It will run for 10 shoppers and then terminate. I plan to alter this so the cart is also a queue loaded from a predetermined list of products.
    View the code at the Mystery Market repo.

  • Potentiometer Sampling on the Microchip PIC16F887 chip
    Done in assembly. The chip continuously polls the output of a potentiometer and then displays the strength in bianry on a 8 element LED DIP package. This is grouped in with the other class assignments mentioned below. Look for the files named "a2d"
    View the code at the MicroChip-PIC16F887-882-assembly repo.
